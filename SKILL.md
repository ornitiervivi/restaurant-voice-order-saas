# Skill — Security

Use esta skill para tarefas com impacto de segurança.

## Regras

- Modelar ameaças proporcionais ao risco.
- Aplicar menor privilégio.
- Validar entradas e proteger saídas.
- Não logar dados sensíveis.
- Conferir autenticação, autorização e auditoria.
- Revisar dependências, configurações e superfícies expostas.
- Registrar riscos residuais.
