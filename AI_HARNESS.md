# AI_HARNESS.md

## Purpose

Define how AI agents must work in this repository.

## Mandatory behavior

- Read SDD files before work.
- Use AGENT_ROLES.md roles.
- Plan before coding.
- Ask blocking questions.
- Keep documentation alive.
- Register decisions, bugs and lessons.
- Validate before delivering when possible.
- Prefer small increments.

## Required output for non-trivial work

- Roles used.
- Files read.
- Plan.
- Blocking questions or assumptions.
- Files changed.
- Validation executed.
- Documentation updated.
