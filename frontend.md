# Compatibilidade — Frontend

## Versões

Definir framework, runtime, package manager e navegadores suportados no projeto derivado.

## Regras

- Preservar acessibilidade.
- Validar responsividade.
- Evitar dependências grandes sem justificativa.
- Manter typecheck, lint e testes configurados quando aplicável.
- Documentar breaking changes visuais ou de contrato.
