# ai-sdd-app-template

Template genérico para aplicações orientadas por **Spec-Driven Development** e apoiadas por um **harness de IA**.

## Objetivo

Fornecer uma estrutura inicial de documentação, compatibilidade e skills para projetos futuros de backend, frontend, banco de dados, segurança e DevOps.

## Importante

Este repositório não implementa um projeto real. Ele contém apenas arquivos-base para iniciar aplicações futuras.

## Começando um projeto derivado

1. Leia `AGENTS.md`.
2. Preencha `PROJECT_CONTEXT.md`.
3. Escreva `SPEC.md` antes de implementar.
4. Defina arquitetura em `ARCHITECTURE.md`.
5. Registre decisões em `DECISIONS.md`.
6. Ajuste compatibilidade em `.compatibility/`.
7. Use as skills em `.skills/` conforme a tarefa.
8. Defina validações em `VALIDATION.md`.

## Estrutura principal

- `AGENTS.md`: contrato central para agentes de IA.
- `SPEC.md`: fonte de requisitos.
- `ARCHITECTURE.md`: visão arquitetural.
- `AI_HARNESS.md`: regras operacionais para IA.
- `.compatibility/`: matrizes e restrições por área.
- `.skills/`: instruções especializadas por tipo de tarefa.
