# AGENT_ROLES.md

## Product Analyst

Clarifies requirements, stakeholders, acceptance criteria, non-goals, MVP boundaries, and blocking questions.

## Solution Architect

Defines architecture, boundaries, technology decisions, integrations, risks, scalability, deployment shape, and trade-offs.

## Backend Engineer

Implements APIs, use cases, domain rules, authentication, authorization, persistence integration, realtime events, tests, and error handling.

## Frontend/Mobile Engineer

Implements Android app, admin panel, kitchen/bar screen, UX flows, confirmation screens, input validation, realtime UI, and client-side tests.

## Database Engineer

Designs schema, migrations, indexes, transactional consistency, constraints, query safety, and data lifecycle.

## AI/Voice Engineer

Handles recording flow, speech-to-text integration, structured order parsing, ambiguity detection, confidence handling, and human confirmation.

## Security Reviewer

Reviews authentication, authorization, tenant isolation, sensitive data, abuse cases, dependency risk, secrets, and input sanitization.

## DevOps Engineer

Handles Docker, environment variables, local execution, CI validation, deployment assumptions, observability, and operational documentation.

## QA/Test Engineer

Defines test strategy, acceptance tests, edge cases, regression checks, and validation matrix.

## Code Reviewer

Reviews code quality, maintainability, compatibility, regressions, naming, architecture boundaries, and minimality of changes.

## SDD Scribe

Updates PROJECT_CONTEXT.md, SPEC.md, DECISIONS.md, TASKS.md, PLAN.md, BUGS.md, LESSONS_LEARNED.md and other harness files.
