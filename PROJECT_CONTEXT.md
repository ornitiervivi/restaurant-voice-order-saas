# PROJECT_CONTEXT.md

## Project name

restaurant-voice-order-saas

## Origin

Workana MVP request for a SaaS system where restaurant waiters create orders through voice commands.

## Objective

Reduce manual typing, order mistakes, communication failures with kitchen/bar, and slow service during peak hours.

## MVP

- Android waiter app.
- Login.
- Table list.
- Open and close orders.
- Voice recording.
- Speech-to-text.
- Structured order parsing.
- Human confirmation before submit.
- Table history.
- Admin panel.
- Product management.
- Table management.
- User/waiter management.
- Open order visualization.
- Kitchen/bar realtime receiving screen.
- Order status updates.

## Critical rule

AI must never submit an order without waiter confirmation.
