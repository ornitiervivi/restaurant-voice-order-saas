# PROJECT_CONTEXT.md

## Nome do template

`ai-sdd-app-template`

## Propósito

Template genérico para iniciar aplicações usando Spec-Driven Development e harness de IA.

## O que este repositório não é

- Não é uma aplicação real.
- Não contém requisitos de produto finais.
- Não deve acoplar stack, domínio ou fornecedor específico sem decisão registrada.

## Como usar

1. Copie este template para um novo repositório de aplicação.
2. Preencha contexto, especificação, arquitetura e validação.
3. Configure skills e compatibilidade para a stack escolhida.
4. Use `AGENTS.md` como contrato operacional para agentes de IA.
